# **App Name**: STUDORA

## Core Features:

- College Management Registration: Allow college management to register their college with subscription fees.
- Student Authentication: Enable students to sign up and select their college.
- Service Marketplace: Students can post requests with service type, description, deadline, and budget.
- Skill Recommendation Tool: An AI tool that suggests optimal skills for providers and service seekers.
- Service Provider Listing: Students can list skills and accept orders.
- Simulated Payments: Simulate payments and track earnings per student.
- Ratings and Reviews: Students can rate each other, with ratings displayed on profiles.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, while still being approachable for students.
- Background color: Very light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Muted green (#4CAF50) to highlight successful transactions or positive reviews.
- Body and headline font: 'PT Sans', a modern humanist sans-serif, suitable for both headings and body text, for a modern yet approachable feel.
- Use simple, outlined icons to represent service categories and profile information.
- A clean, card-based layout for service listings, with clear distinctions between requests and providers.
- Subtle transitions and loading animations to enhance user experience.
